A comprehensive, web-based Vehicle Rental Management System developed for a local agency (Hill Country Cabs). This system transitions traditional paper-based rental operations into a secure, centralized digital platform. 

It features a customer-facing booking portal with real-time dynamic pricing, a chatbot and a secure administrative backend for fleet, revenue, and booking management.

---

## ✨ Key Features

### 👤 Customer Portal
* **Secure Authentication:** User registration and login with encrypted passwords (`bcrypt`).
* **Interactive Fleet Browsing:** Filter and search available vehicles (Cars, Vans, SUVs) with real-time availability checking to prevent double-booking.
* **Dynamic Booking Engine:** JavaScript-powered real-time cost estimation factoring in rental duration, base rates, and optional driver fees.
* **Booking Dashboard:** Customers can track their reservation status (Pending, Confirmed, Completed) and view digital invoices.
* **Automated Support:** Integrated chatbot for 24/7 automated customer inquiries.

### 🛡️ Admin Dashboard (RBAC)
* **Fleet Management (CRUD):** Add, update, or remove vehicles, including secure image uploads.
* **Booking Workflow:** Approve or reject pending reservations and assign specific drivers.
* **Return Processing Engine:** Automated calculation of excess mileage penalties and manual entry for vehicle damage charges upon return.
* **Analytics & Reporting:** Real-time data aggregation displaying total revenue, active rentals, and overall business performance metrics.

---

## 🛠️ Technology Stack

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla/ES6), Bootstrap 5.
* **Backend:** PHP (Procedural & Object-Oriented paradigms).
* **Database:** MySQL (Normalized to 3NF).
* **Architecture:** LAMP Stack (Linux/Windows, Apache, MySQL, PHP).
